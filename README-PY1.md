# python-memo-cli　　（生成AI利用）

PythonとSQLiteで作成したCLIメモアプリです。

## 機能
- メモ追加
- メモ一覧表示
- メモ削除

## 使用技術
- Python
- SQLite3

# CLIメモアプリの動作確認

AIにCLIメモアプリのコードを書いてもらい、自分で実行しながら動作を理解しました。create_table / add_memo / get_memos / delete_memo の各関数の中で、データベース接続→SQL実行→commit→close という一連の流れを1つずつ確認しています。


## 学んだこと
- Pythonの関数分割
- 別ファイルのimport
- SQLiteでのデータ保存
- Git/GitHubでの管理
